# Netflix Clone

This project is a clone of the popular streaming service Netflix, created as a part of my web development learning journey. The aim of this project was to replicate the main features and layout of Netflix, allowing users to browse TV shows and movies, sign in, and access various sections of the platform.

## Features

- **Main Page:** The landing page of the website showcases a hero section with a call-to-action to join the platform. Users can enter their email address to create or restart their membership.
- **Browse Content:** Users can browse through different sections of the website to learn about features such as watching on TV, downloading shows for offline viewing, and streaming content on multiple devices.
- **Frequently Asked Questions:** The FAQ section provides answers to common queries users might have about Netflix, including information about pricing, available content, and supported devices.
- **Footer:** The footer contains links to various resources such as investor relations, help center, media center, and contact information.

## Technologies Used

- **HTML:** Used for structuring the web pages and creating the layout.
- **CSS:** Utilized for styling the website, including colors, fonts, and layout.

## Setup and Usage

1. Clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Explore the different sections of the website and enjoy browsing!
